Summary :

 * FAPI Certificate
   Keycloak 15 has achieved certificates for FAPI 1.0 Advanced (all 8 conformance profiles), FAPI-CIBA (all 4 conformance profiles) and Open Banking Brazil FAPI 1.0 (8 of 9 conformance profiles).

 * OIDC Certificate 
   Keycloak will re-acquire OIDC OP certificate. It will also acquire OIDC Logout proflie certificate.

 * Automated conformance test environment for OIDC and its Logout profile certificate
   It is helpful if someone support automated conformance test environment in FAPI-SIG repository for OIDC and its Logout profile certificate.

 * Automated conformance test environment against keycloak-x
   It is helpful if someone support automated conformance test environment in FAPI-SIG repository against quarkus based keycloak-x.
   Currently conformance test run against keycloak based on wildfly.

 * RAR/Grant Management API
   It is good if a red hat's developer working on dynamic scopes could participate FAPI-SIG meeting. We might help him.

 * DPoP
   The draft version PR for DPoP is in progress.

 * Keycloak 17
   Keycloak 17 based on quarkus may be released in Feb or Mar.
