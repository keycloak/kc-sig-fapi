Summary :

 * Updates : keycloak
   Keycloak 14 has been released.
   https://www.keycloak.org/2021/06/keycloak-1400-released.html

 * Updates : FAPI 1.0​
   Keycloak 14 officially supported FAPI 1.0 Baseline/Advanced support as default client profile.​
   FAPI-SIG’s automated conformance test run environment provided FAPI1 Advanced Final conformance tests against keycloak 14 as default.

 * Updates : FAPI 2.0​
   PR for OAuth 2.0 Rich Authorization Requests (RAR) has been sent to keycloak upstream, waiting for its review.
   https://github.com/keycloak/keycloak/pull/8006

 * Updates : Client Policies​
   Keycloak 14 officially supported Client Policies.

 * New item : FAPI-JARM
   PR for FAPI JWT Authorization Response Mode (FAPI-JARM) has been sent to keycloak upstream, waiting for its review.
   https://github.com/keycloak/keycloak/pull/8158

 * Updates : SPA/Native App​
   There is a progress on frontend side (js).

 * Other Discussion Topics
   - keycloak 14 might try to get the certificates for "FAPI Adv. OP w/ MTLS" and "FAPI Adv. OP w/ Private Key" within about two weeks or so.
   - Which security profiles we try to support after FAPI 1.0 support is completed.
     -> For the time being, PRs for PAR, FAPI-CIBA and FAPI-JARM could be hopefully reviewed by keycloak development team.
   - Proposal : apply for CfP to DevConf.cz 2022 for presenting FAPI-SIG activities and achievements to OSS developer’s community.
