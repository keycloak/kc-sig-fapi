Summary :

 * Governance
   FAPI-SIG needs its clear governance.
   Defining clear goals and projects to achieve them
   A few people manage and coordinate these projects
   Assigning contributors to them and watch their progresses
   Clarifying contributors of FAPI-SIG
   Taking notes about regular meetings

 * Ongoing works
   Banfico has made progresses on running the latest FAPI-RW conformance tests and tries to give feedback to making them onto CI/CD pipeline of keycloak.

 * Client Policies
   Security profile like FAPI-RW relies on Client Policies.
   However, Client Policies is still preview feature. Their UI is still missing.

 * New Admin REST API (clear JSON Documentation)
   New clear JSON Documentation will become introduced (e.g. User Profile).
   It affects Client Policies related works.

 * New Admin Console
   New Admin Console and its UIs will be introduced.
   It affects Client Policies related works.
