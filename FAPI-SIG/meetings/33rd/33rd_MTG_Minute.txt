Summary :

 * OIDC Certificate 
   Keycloak will re-acquire OIDC OP certificate for the following conformance profile:
   - Config OP
   - Basic OP
   - Implicit OP
   - Hybrid OP
   - Dynamic OP
   - FORM POST OP
   Currently, the remaining 3rd Party-Init OP conformance profile is out of scope.
   
 * OIDC Certificate for logout profiles
   Keycloak will newly acquire OIDC OP certificate for the following conformance profile:
   - RP-Initiated OP
   - Session OP
   - Front-Channel OP	
   - Back-Channel OP

 * Automated conformance test environment for OIDC
   It has supported by the current FAPI-SIG's automated conformance test environment.
 
 * Automated conformance test environment for OIDC Logout profile certificate
   It has not yet supported by the current FAPI-SIG's automated conformance test environment.

 * Automated conformance test environment against quarkus based keycloak
   It has supported by the current FAPI-SIG's automated conformance test environment.
