Summary :

 * OID4VCI
   Epic issue: https://github.com/keycloak/keycloak/issues/25936
   22 issues of 21 issues in total were resolved (+5 new issues, +4 resolved).
   Experimental support of OID4VCI was completed.
   We continue working OID4VCI support to mature this feature.

 * Token Excnahge
   Epic issue: TBD
   SPI provider for authorization of Token Exchange was proposed.
   We might have a breakout session.

 * DPoP
   Epic issue: https://github.com/keycloak/keycloak/issues/22311
   3 issues are open. No progress. Currently a preview feature.
   We might have a breakout session. (some member of OAuth SIG can work on promoting DPoP as official feature)

 * Passkey
   Epic issue: https://github.com/keycloak/keycloak/issues/23656
   6 issues of 11 issues in total were resolved. No progress. Currently a preview feature.

 * OpenID Connect Native SSO for Mobile Apps 1.0
   Spec (Implementer's Draft): https://openid.net/specs/openid-connect-native-sso-1_0.html
   It was proposed to support as one use-case of Token Exchange.

 * OpenID Federation 1.0 (OIDFED)
   Spec (Imprementer's Draft): https://openid.net/specs/openid-federation-1_0.html
   One of a use-case of OIDFED is GAIN POC (https://openid.net/cg/gain-poc/)
   There are community works of OIDFED for Keycloak.
     https://github.com/eosc-kc/keycloak-oidc-federation (Seems to be based on an early darft version (13))
     https://github.com/keycloak/keycloak-community/pull/201
     https://github.com/thomasdarimont/keycloak-oidc-federation/tree/update/keycloak-25.0.x
   Sphereon's Swagger API for OIDFED endpoints
     https://app.swaggerhub.com/apis/SphereonInt/OpenIDFederationAPI/1.0.0-d35
   We continue watching these works.

 * Keycloak Spec Sheet
   How about creating document showing which specification Keycloak has already supported?
   E.g., https://www.thecyberhut.com/oauth2-capabilities-matrix-vendor-comparison/
